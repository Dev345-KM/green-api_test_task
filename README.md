# green-api_test_task

Сервис для проверки работы GREEN-API 
Написанный на HTML & Python при помощи библиотеки FastAPI


 ## Для теста можно запустить локально с помощью Docker
_введите в терминале команды_

 <pre>
 Bash

docker build -t green-api-app .
docker run -p 8000:8000 green-api-app

</pre>
